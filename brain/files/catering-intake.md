# Catering Intake + Follow-up Script

## Required fields
1. Event date
2. Event time
3. Location
4. Guest count
5. Service style
6. Dietary requirements
7. Budget or package preference
8. Contact info

## First response style
Friendly, fast, specific. Confirm the request, ask for missing fields, and promise a follow-up owner.

## Escalation
Any lead with date + guest count + contact info should be marked high priority until owner confirms follow-up.
